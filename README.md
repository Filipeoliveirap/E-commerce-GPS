# Mini Ecommerce

## Tecnologias
- Backend: Java Spring Boot
- Frontend: React JS

## Como rodar
### Backend
cd backend
./mvnw spring-boot:run  # ou ./gradlew bootRun

### Frontend
cd frontend
npm install
npm start

## Estrutura de pastas
- backend/: projeto Spring Boot
- frontend/: projeto React JS

## Colaboração
- Branch por funcionalidade
- PR para main
- Commits claros
